# Ember Tide

**Tagline:** *Rekindle the light before the world forgets it ever shone.*

**Genre:** Action-Platformer / Metroidvania-lite

---

## Summary

Ember Tide is a Game Boy Color platformer built in GB Studio 4.2. You play as Solen, a young lamplighter from the village of Cinderhearth, who sets out across the fractured world of Lumara to reignite the five ancient Hearthstones — massive crystalline beacons whose dying light is the only thing holding back the Hollow Tide, a wave of living darkness consuming the land. Solen's journey is personal: her mother, Maren, was turned to stone by the encroaching dark, and only the full light of all five Hearthstones can reverse the petrification. Armed with her mother's lantern staff and a stubborn refusal to give up, Solen must traverse forests choked with shadow, frozen peaks, sunken ruins, volcanic forges, and a sky citadel hanging above the void — battling corrupted creatures, solving environmental puzzles, and uncovering the truth about the Hollow King, the fallen guardian who extinguished the Hearthstones in the first place. With tight platforming, chiptune music, and a story told through pixel art and dialogue portraits, Ember Tide is a love letter to the golden age of handheld gaming.

## Technical Specifications

| Spec | Detail |
|---|---|
| Engine | GB Studio 4.2 |
| Export Format | Game Boy Color ROM (.gbc) |
| Resolution | 160×144 pixels |
| Tile Size | 8×8 px |
| Scene Type | PLATFORMER |
| Color Mode | GBC — 8 BG palettes, 8 sprite palettes, 4 colors each |
| Max Actors/Scene | 20 (max 10 visible on screen) |
| Max BG Size | 32×32 tiles (256×256 px) |
| ROM Limit | 4 MB |
| Sprite Sizes | 16×16 standard · 16×32 tall · 8×16 small |
| Music Format | hUGETracker, 4-channel GB audio |
| Dialogue | Display Dialogue event, 16×16 portrait + text box |
