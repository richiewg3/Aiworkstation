# Asset Prompts — Master List

This document compiles every pixel art generation prompt from the entire Ember Tide Game Design Document into a single organized reference. Use these prompts with pixel art AI tools or as specifications for manual pixel art creation.

All assets use GBC-style indexed color with 4-color palettes, no anti-aliasing, and clean pixel art style.
